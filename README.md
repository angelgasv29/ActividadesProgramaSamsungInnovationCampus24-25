# Portafolio de actividades y proyectos realizados - Samsung Innovation Campus (SIC):

Se inicia con la carpeta (SIC), la cual contiene el primer módulo de programación y a partir de la semana 11
el segundo módulo de Inteligencia Artificial. Cada carpeta se identifica la semana de formación, donde se exploran
diversos temas, avanzando gradualmente su complejidad, hasta poder articular los conocimintos necesarios para la realización
del proyecto de final de módulo, el cual certifica de forma completa los diversos temas y habilidades.
